# 비트마스킹 
컴퓨터에서 자료를 표현하기 위해 비트를 사용합니다.

1bit = 0 또는 1

8bits = 1byte
비트 연산자
다음은 6가지 비트 연산자입니다.

비트 연산자	a = 0b1010, b = 0b0100
&	AND	a & b = 0b0000
|	OR	a | b = 0b1110
^	XOR	a ^ b = 0b1110
~	NOT	~a = 0b0101
<< 	왼쪽 Shift	a << n = a * 2n
>> 	오른쪽 Shift	a >> n = a * 2-n
